How to use this architecture "siki"

1.This architecture needs magic number x"ffffffff" at the end of the instruction you want to load.

2.This architecture will print all of the data stored in every register when every instruction has executed.


NOTES

1.This is a proto type of 1st architecture. This cannot set beggning PC or floating points.

2.This has not implemented system call yet.

3.The origin of this architecture's name is the Buddist concept of the distinct called Vijñāna, or "siki" in Japanese.
